## Official grammars

### Parsers for these languages are fairly complete:

  * Bash
  * C
  * C++
  * CSS
  * ERB / EJS
  * Go
  * HTML
  * JavaScript
  * OCaml
  * PHP
  * Python
  * Ruby
  * Rust
  * TypeScript

### Parsers for these languages are in development:

  * Agda
  * C-sharp
  * Haskell
  * Java
  * Julia
  * Scala
  * [verilog](https://github.com/tree-sitter/tree-sitter-verilog)
